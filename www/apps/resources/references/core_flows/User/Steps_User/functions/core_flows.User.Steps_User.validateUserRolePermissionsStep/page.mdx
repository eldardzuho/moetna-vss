# validateUserRolePermissionsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `validateUserRolePermissionsStep`. It belongs to the `@moetnavss/medusa/core-flows` package.

Validates that the actor has all the policies from the roles being assigned.
A user can only assign roles whose policies they themselves have.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@moetnavss/framework/workflows-sdk"
import { validateUserRolePermissionsStep } from "@moetnavss/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = validateUserRolePermissionsStep({
      "actor_id": "{value}",
      "role_ids": [
        "{value}"
      ]
    })
  }
)
```

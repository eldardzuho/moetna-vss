# assignUserRolesWorkflow - Medusa Core Workflows Reference

This documentation provides a reference to the `assignUserRolesWorkflow`. It belongs to the `@moetnavss/medusa/core-flows` package.

This workflow assigns roles to users.
Supports two modes:
- Assign multiple roles to a single user: &#123; user\_id, role\_ids &#125;
- Assign multiple users to a single role: &#123; user\_ids, role\_id &#125;
It validates that the actor has all the policies from the roles being assigned.

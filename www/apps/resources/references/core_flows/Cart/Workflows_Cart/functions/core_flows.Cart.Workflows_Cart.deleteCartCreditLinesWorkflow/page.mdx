# deleteCartCreditLinesWorkflow - Medusa Core Workflows Reference

This documentation provides a reference to the `deleteCartCreditLinesWorkflow`. It belongs to the `@moetnavss/medusa/core-flows` package.

This workflow deletes one or more credit lines from a cart.

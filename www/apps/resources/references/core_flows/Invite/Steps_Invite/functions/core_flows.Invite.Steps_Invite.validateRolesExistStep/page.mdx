# validateRolesExistStep - Medusa Core Workflows Reference

This documentation provides a reference to the `validateRolesExistStep`. It belongs to the `@moetnavss/medusa/core-flows` package.

This step validates that the provided role IDs exist in the RBAC module.
Throws an error if any role is not found.

## Example

```ts
validateRolesExistStep(["role_123", "role_456"])
```
